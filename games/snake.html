Pixel Art Editor

Создавайте потрясающие пиксельарты прямо в браузере

Мощный редактор пиксельной графики с множеством инструментов для творчества. Идеально для создания спрайтов, иконок и артов.

Инструкция по использованию

  1. Каждое приложение находится в отдельной папке: snake/ и pixelart/
  2. Нажмите на кнопку под интересующим вас приложением для перехода к нему
  3. Для игры в "Змейку" используйте стрелки на клавиатуре или кнопки на экране (на мобильных)
  4. В Pixel Art Editor используйте различные инструменты для создания пиксельной графики
  5. Все приложения работают полностью в браузере и не требуют установки
  6. Для возврата в меню нажмите кнопку "Меню" в левом верхнем углу
